## Basic Git Commands


```bash
git init	                                Initialize a new Git repository
git add .	                                Add all files and folders to the staging area
git commit -m "message"	                    Commit changes with a descriptive message
git remote add origin <URL>	                Add a remote repository URL
git push -u origin branch-name	            Push changes to the remote repository
git clone <URL>	                            Clone a repository from GitHub to your local machine
git pull origin branch-name	                Fetch and merge changes from the remote repository
git remote -v	                            View the remote repository URLs
